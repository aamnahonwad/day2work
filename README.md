# Day 2 – Java Methods, Classes & Objects

This repository contains Java programs implemented as part of Day 2 lab work.  
The focus was on understanding Java methods, arrays, classes, objects and basic object-oriented concepts.

---

## Section 1 – Methods & Basic OOP

### Topics Covered
- Creating and calling methods
- Passing parameters to methods
- Returning values
- Working with arrays
- Introduction to classes and objects
- Encapsulation (using private variables with getters and setters)
- Basic validation using setter methods

### Programs Implemented
1. Add Two Numbers using Method
2. Find Maximum using Method
3. Prime Number Check
4. Pass Array to Method
5. Student Class
6. Rectangle Class
7. Bank Account Class
8. Employee Class (Encapsulation)
9. Car Class with Validation
10. Product Class with Constructor and Getters/Setters

---

## Section 2 – Intermediate Methods & Classes

### Topics Covered
- Loops inside methods
- String manipulation
- Method Overloading
- Constructors
- Object creation and usage
- Data validation using setters

### Programs Implemented
1. Calculate Power using Loop
2. Reverse String
3. Count Vowels
4. Method Overloading – Area Calculation
5. Book Class
6. Circle Class
7. Laptop Class
8. Person Class with Age Validation
9. ATM Class with Balance Check
10. College Class using Constructor
